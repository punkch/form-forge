# Golden files

Generated by scripts/make-goldens.py with **pyxform 4.5.0**.

`src/*.xlsx` are the XLSForm fixtures; `expected/*.xml` is pyxform's
output for each, which our serializer tests compare against
(canonicalized, with a documented divergence allowlist).

Regenerate deliberately with:

    uv run --with openpyxl --with pyxform scripts/make-goldens.py

and review the diff before committing.
